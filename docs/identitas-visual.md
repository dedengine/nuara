# Identitas Visual Nuara

Identitas visual Nuara menggabungkan piring makanan sehat, tunas, dan bentuk tangan pelindung. Maknanya adalah pertumbuhan anak, pemenuhan nutrisi, dan pengawasan makanan dari dapur sampai sekolah.

## Warna

| Peran | Warna | Hex |
|---|---|---|
| Utama | Hijau teal | `#087E6C` |
| Aksen hangat | Oranye | `#F29F3D` |
| Aksen informasi | Biru | `#3274D9` |
| Latar lembut | Hijau muda | `#E4F3EF` |
| Latar aplikasi | Abu hijau sangat muda | `#F4F7F6` |

## Mode Gelap

Web Admin dan aplikasi Mobile menyediakan pengalih mode terang/gelap dan menyimpan pilihannya pada perangkat. Mode gelap mempertahankan aksen teal dan oranye, dengan teal terang `#63D6BE`, aksen hangat `#FFB45A`, latar `#101714`, permukaan `#18221F`, dan garis pemisah `#3A4A45` agar kontrol, teks, tabel, dan kartu tetap mudah dibaca.
